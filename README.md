# 🔒 ShieldPass - Advanced Password Strength Analyzer

![ShieldPass](https://img.shields.io/badge/ShieldPass-Password%20Analyzer-00ff9d)
![Version](https://img.shields.io/badge/version-2.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

A modern, interactive, **client-side** password strength analyzer with real-time security evaluation, breach detection, and smart recommendations.

## 🚀 What's New in v2.0

- **Crack Time Estimation** - Realistic estimates of how long it would take to crack your password
- **Password History** - Local history of analyzed passwords (last 8) with one-click reload
- **Smart Password Generator** - Customizable generator + memorable passphrase mode
- **Common Password Detection** - Warns if your password is among the most commonly used
- **Copy Analysis Report** - One-click copy of a full security report
- **Improved Scoring & Recommendations** - Smarter analysis and more actionable advice
- **Better UI/UX** - Enhanced visuals, animations, and mobile experience

## 🚀 Original Features

- **Real-time Strength Analysis** - Visual password strength meter with instant feedback
- **Character Diversity Check** - Comprehensive analysis of password composition
- **HaveIBeenPwned Integration** - Check if password appears in known data breaches (k-anonymity)
- **Entropy Calculation** - Mathematical measurement of password randomness
- **Security Recommendations** - Contextual suggestions for improving password security
- **Dark/Light Mode** - Toggle between dark and light themes
- **100% Client-side** - All analysis happens locally in your browser. No data is sent to any server.

## 🛠️ Tech Stack

- **HTML5** - Semantic markup and structure
- **CSS3** - Modern styling with CSS variables, glassmorphism, and animations
- **JavaScript ES6+** - Interactive functionality and secure API integration
- **HaveIBeenPwned API** - Privacy-preserving password breach checking

## 🔧 How to Use

1. Open `index.html` in any modern browser (or visit the GitHub Pages link if deployed)
2. Type or paste a password
3. Instantly see strength, entropy, breach status, and personalized recommendations
4. Use the generator to create strong passwords or passphrases
5. Review your analysis history

## 🔒 Privacy & Security

- Everything runs **locally** in your browser
- Passwords are **never sent** to any server (except the privacy-preserving k-anonymity check to HaveIBeenPwned)
- History is stored only in your browser's localStorage
